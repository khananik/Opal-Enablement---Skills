Anatomy of a Skill

A properly configured skill consists of several components defining its behavior and trigger conditions.

1 Name

A short identifier for easy recognition.

2 Active Switch

A master toggle to enable or disable the skill.

3 Shortcut

A keyword starting with / (e.g., /brand-voice) to invoke in chat.

4 When to Use

Activation trigger via keywords, intent, or tool runs.

5 The Core Skill

The prompt payload. Write it as if instructing a new team member:

  • Objectives: What it achieves.
  • Steps: Processing instructions.
  • Format: Expected layout.
  • Constraints: Safety measures.

6 Files

Reference documents Opal reads when applying the skill.

7 Where to Use

Specifies product instances where the skill activates.

8 Required Tools

Tools that must be available for it to function.

Activation & Relevancy

How Opal decides which skills to trigger and how to scope keywords.

The Selection Process

Opal follows a four-step process:

1

Collection

Gathers all active skills available to the user.

2

Tool Matching

Applies skills triggered by specific required tools.

3

Relevancy Evaluation

Opal reads the latest message to find semantic matches. It prioritizes inclusion over exclusion.

4

Context Integration

Matched skills are injected into the system prompt.

Scoping Keywords

Broad descriptions cause over-triggering. Use explicit boundaries.

GoalTechniqueExample
Exact topics"ONLY use for...""ONLY use when user mentions A/B test"
Prevent false matches"Do NOT use for...""Do NOT use for general analytics"
Anchor to featureName the tool"When configuring promotion rules..."
Handle synonymsList alternatives"Content scoring, grades, quality scores"

Managing & Maintaining Skills

Modifying Skills

Open any skill by clicking its name or More (...) > Edit.

Best Practices

  • Preserve Purpose: Don't remove core structures without equivalents.
  • Be Explicit: Use imperative language.
  • Maintain Logic: Keep conditional logic intact.
  • No UI References: Avoid "Click here" language.

Delete from Dashboard

Click More (...) > Delete.

Delete from Editor

Click More (...) > Delete Skill.
